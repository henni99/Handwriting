HandwritingController

@Stable
class HandwritingController

Controller for managing handwriting data, such as pen strokes, eraser actions, and lasso selection. It supports undo/redo functionality, tool modes, and manages the state for various elements like strokes, selected paths, and the lasso area.

fun selectHandWritingPath(path: Path)

Selects handwriting paths that are within or overlapping the given path, typically used for lasso selection. The function first calculates the bounding box of the provided path and checks each handwriting path in the handwritingPaths collection to see if its hit area overlaps with or is completely contained within the bounding box of the lasso selection.
